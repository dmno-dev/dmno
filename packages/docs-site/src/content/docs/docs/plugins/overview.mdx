---
title: Plugins
---

DMNO provides 1st-party plugins to securely store (and retrieve) your application secrets.


- [**DMNO Encrypted Secrets**](/docs/guides/plugins/encrypted-vault): Store secrets securely in an encrypted vault that you check in to your git repo. 
- [**DMNO 1Password Sync**](/docs/guides/plugins/1password): Securely sync secrets with 1Password 


:::note

We will soon be opening up our APIs for developers to create their own DMNO plugins. If you'd like to be part of that, drop us a line via <a href={`mailto:${DMNO_PUBLIC_CONFIG.GENERAL_CONTACT_EMAIL}`} target="_blank">email</a> or join our <a href={DMNO_PUBLIC_CONFIG.DISCORD_JOIN_URL} target="_blank">Discord</a>.
:::
