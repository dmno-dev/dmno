<CliCommand name="run" />

:::caution
While `dmno run -- echo $SOME_ITEM` seems like it should work, your shell will try to resolve that variable _before_ dmno injects your resolved config as env vars.

Instead you must use `printenv` - for example `dmno run -- printenv SOME_ITEM`.
:::
