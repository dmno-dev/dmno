The `dmno-cli` was installed when you [installed](/reference/config-engine/installation) `dmno` and is available as a depedency in your project. 

The `dmno-cli` provides a number of commands to help you run and manage your DMNO projects.

### Commands

- [`dmno init`](/reference/cli/init) - Initialize a new DMNO project
- [`dmno load`](/reference/cli/load) - Load the resolved config for a service
- [`dmno dev`](/reference/cli/dev) - Run a service in dev mode, watching for changes and updates
- [`dmno run`](/reference/cli/run) - Run a command with the resolved config for a service
- [`dmno plugin`](/reference/cli/plugin) - Run plugin specific commands
